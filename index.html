離線下載

若您想下載所有簡報並稍後觀看，您可以使用以下方式下載。

  1. 藉由 Git 下載 
    git clone git://github.com/chusiang/rvl.js.git
cd rvl.js/
git checkout gh-pages
firefox index.html
  2. 藉由 wget 下載 
    wget https://github.com/chusiang/rvl.js/archive/gh-pages.zip
unzip gh-pages.zip 
firefox rvl.js-gh-pages/index.html
